GitHub

Look at the source code for this website at GitHub.

Editor

A text editor used to edit files on this website. This feature is available only to users with a login.

Post Tools

Send post requests to other pages on this website. This tool is unlikely to be very useful to anyone who does not have a login. Additionally, an intuitive user interface has not yet been added.

Trailblazer

A tool used to facilitate an old, cumbersome board game called Trailblazer. It keeps track of the goods produced on each star, and calculates demand, etc. The physical game is still required to play, however.

Dungeons and Dragons is not the only game we run; We mostly play Pathfinder, but other Role-Playing Games (RPGs) can be run if enough people are interested. We run games for all skill levels from complete beginners to experts.

Recommended materials:

  • Pencil and Scratch Paper
  • Dice
    • One set contains 1d4, 1d6, 1d8, 1d10, a percentile die, 1d12, and 1d20.
    • Dice are typically available for trade, borrowing, and purchasing by individuals during lunch on tuesdays.

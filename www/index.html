Snake

Eat the blue squares to grow longer. Use the arrow keys to move while avoiding running into the walls or yourself.

The current verified high score is 590.

Paint

A game based off of capture the flag. Instructions are included.

Game

Using the arrow keys to move, avoid the red rectangles, using the green powerups to help you on your way. A second player can use WASD to have an invincible blue rectangle slowly chase after player 1.

Game - 3d

Adds a whole new dimension to Game! Use the arrow keys to move, space to restart. Try not to be hit by the TNT. Please not that this game especially is far from complete.
